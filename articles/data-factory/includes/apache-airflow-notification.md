> [!IMPORTANT]
> On January 1, 2026 you'll no longer be able to create new Airflow instances using ADF's Workflow Orchestration Manager. We recommend that you migrate all Workflow Orchestration Manager (Apache Airflow in Azure Data Factory) workloads to [Apache Airflow jobs in Microsoft Fabric](/fabric/data-factory/apache-airflow-jobs-concepts) before December 31, 2025.
>
> For more information or for support during your migration to Apache Airflow in Microsoft Fabric, contact Microsoft Support.
